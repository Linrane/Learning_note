## 第9节

容器里面套容器形成的嵌套结构：

```python
students = [
    {"name": "张三", "score": 88},
    {"name": "李四", "score": 92},
    {"name": "王五", "score": 75},
]
# 取第1个学生的成绩
students[0]["score"]        # 88
# 遍历所有名字
for s in students:
    print(s["name"])
# 找最高分学生
best = max(students, key=lambda s: s["score"])
```

（这是列表套字典的现象）

字典套列表的也有：

```python
classes = {
    "一班": ["张三", "李四"],
    "二班": ["王五", "赵六", "钱七"],
}
# 二班有多少人
len(classes["二班"])        # 3
# 遍历每班人数
for cls, names in classes.items():
    print(f"{cls}有{len(names)}人")
```

字典套字典的都有：

```python
info = {
    "张三": {"age": 20, "city": "北京"},
    "李四": {"age": 22, "city": "上海"},
}
info["张三"]["city"]        # "北京"
```

易错点：

- `students[0]["score"]` 是两层访问：先 `[0]` 取列表元素（字典），再 `["score"]` 取字典值
- 嵌套取值要一层层来，不能跳过
